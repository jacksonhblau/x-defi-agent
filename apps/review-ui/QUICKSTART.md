# Quick Start Guide - DeFi Agent Dashboard

## ⚡ 30-Second Setup

```bash
cd apps/review-ui
npm install
npm run dev
```

Open http://localhost:3000

## 📝 What You Get

A complete, production-ready admin dashboard with 8 pages for managing DeFi posts:

- 📋 **Drafts** - Manage and approve post drafts
- 📅 **Calendar** - Visual timeline of posts
- 📊 **Posts** - Performance analytics
- 📰 **Stories** - Published content tracking
- ⚡ **Signals** - System alerts
- 💼 **Jobs** - Background job management
- ⚙️ **Config** - System settings
- 👀 **Watchlist** - Token monitoring

## 🔑 Key Features

✅ Full authentication with NextAuth.js  
✅ Responsive design (mobile-friendly)  
✅ Real-time data updates  
✅ TypeScript for type safety  
✅ Production-ready code  
✅ Zero external component libraries  
✅ Built with Next.js 16 & Tailwind CSS 4  

## 🚀 Next Steps

1. **Set env vars** in `.env.local`
2. **Connect Supabase** (optional) for database
3. **Deploy to Vercel** or your preferred host
4. **Customize** colors, auth, and features
